# HabitFlow 🚀

HabitFlow is a premium, minimal habit tracker designed for a seamless and distraction-free experience. Built with performance and aesthetics in mind, it helps you build lasting routines with ease.

## 🔗 Live Demo
**[Launch HabitFlow App](https://protikchowdhury.github.io/Habitflow/)**

---

## ✨ Key Features

- **☁️ Optional Cloud Sync**: Sign in with Email or Google to sync your habits across devices using Firebase.
- **⚡ Local-First Design**: Works instantly offline and falls back to Local Mode if synchronization is unavailable.
- **📦 Habit Archiving**: Keep your dashboard clutter-free by archiving completed or paused habits without losing their history.
- **⏳ Undo Deletion**: Accidentally deleted a habit? You have a 5-second window to bring it back.
- **📅 Visual Progress**: Track your consistency with an integrated calendar view and completion heatmaps.
- **🎨 Premium Aesthetics**: A modern dark-mode interface featuring glassmorphism, smooth animations, and Inter typography.

---

## 🛠 Tech Stack

- **Logic**: Vanilla JavaScript (ES6+)
- **Styling**: Modern CSS3 with Custom Properties (Variables)
- **Backend**: Firebase Auth & Firestore (Optional)
- **Icons/Fonts**: Google Fonts (Inter)

## 🚀 Getting Started

### Prerequisites
To run this project locally, you only need a web browser.

### Running Locally (macOS)
1. Clone the repository:
   ```bash
   git clone https://github.com/protikchowdhury/Habitflow.git
   ```
2. Navigate to the folder and double-click `start_app.command` or run:
   ```bash
   ./start_app.command
   ```

### Running Locally (Other OS)
If you don't want to use the script, simply serve the folder using any HTTP server:
```bash
# Example using Python
python3 -m http.server 8000
```
Then open `http://localhost:8000` in your browser.
